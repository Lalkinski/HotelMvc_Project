## 📌 Project Description

HotelMvc is a web application built with ASP.NET Core MVC that provides a complete system for hotel and reservation management.  
The application allows users to browse available hotels and rooms, make reservations, and manage their bookings, while administrators have access to a dedicated admin area for managing system data.

The system follows a clear layered architecture, separating responsibilities between presentation, business logic, and data access layers, ensuring maintainability, scalability, and clean code structure.

Users can:
- browse hotels and rooms
- search by name or location
- create reservations
- view their reservations

Administrators can:
- create hotels and rooms
- view all users (guests)
- manage reservations
- delete reservations

The project includes proper data validation, protection against common security vulnerabilities, and custom error pages (404 and 500), making it robust and reliable.

Additionally, unit tests are implemented for the business logic layer, ensuring correctness of the core system functionality.
